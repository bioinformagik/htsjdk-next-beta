== Getting started

=== How to build

* Linux/MacOS
+
....
./gradlew assemble
....
* Windows
+
....
.\gradlew.bat assemble
....

=== How to test

* Linux/MacOS
+
....
./gradlew check
....
* Windows
+
....
.\gradlew.bat check
....
